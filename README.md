# 🔹 Moe Hub - Kill Noobs Simulator Script
## A powerful, feature-packed script for Kill Noobs Simulator with ESP, Teleports, and Custom Scripts.

 ![image](https://i.ibb.co/gbBvnhTT/Screenshot-2025-02-27-000919.png)

#

📌 Features

✅ ESP System (Crates, Upgrades, Safe Boxes, Cars)

✅ Instant Teleports (Nearest Crate, Upgrade, Safe Box, Car, Safe Zone)

✅ Custom Scripts (Infinite Ammo, Glock Switch, Hitbox Expander, Azure Modded, Infinite Yield)

#

🎮 How to Use

1️⃣ Copy & execute the script in your preferred Roblox executor.

2️⃣ Press RightControl to open/close the GUI.

3️⃣ Use the "Main" tab for ESP toggles and custom scripts.

4️⃣ Use the "Teleport" tab for teleport options.
